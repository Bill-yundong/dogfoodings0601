## 1. 产品概述

HomeAutoPulse 是一套基于 Vue 3 的家庭自动化逻辑冲突监控与解析平台，专注于解决多维度传感器触发数据在安防系统与智能家居控制中心之间的语义对齐问题，通过异步冲突解析引擎优化联动闭环，利用 IndexedDB 存储智能设备离线工况快照，保障复杂场景下全屋智能的逻辑可靠性。

- 核心目标：消除安防与家居系统间的逻辑冲突（如「安防布防模式」与「智能灯光迎宾模式」同时触发），实现多源异构传感器数据的语义级统一
- 目标用户：智能家居集成商、全屋智能运维工程师、IoT 平台开发者

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 系统管理员 | 本地初始化 | 全局配置、规则管理、冲突仲裁策略调整 |
| 运维工程师 | 本地账户 | 监控面板查看、快照管理、告警响应 |

### 2.2 功能模块

1. **实时监控仪表盘**：全局设备状态总览、冲突热力图、传感器数据流实时可视化
2. **冲突解析中心**：异步冲突队列、语义对齐映射表、优先级仲裁规则编辑器、联动闭环追踪
3. **设备快照管理**：离线工况快照存储/恢复、快照对比分析、设备状态时间线
4. **规则引擎配置**：触发条件定义、联动动作编排、冲突策略模板、语义映射规则

## 3. 核心流程

用户通过实时监控仪表盘感知冲突 → 冲突进入异步解析队列 → 语义对齐引擎将安防/家居异构数据映射为统一语义 → 仲裁规则引擎根据优先级策略解析冲突 → 解析结果驱动联动动作执行 → 执行结果反馈至闭环追踪 → 设备状态变化写入 IndexedDB 快照

```mermaid

    "传感器触发" --> "数据采集层"
    "数据采集层" --> "语义对齐引擎"
    "语义对齐引擎" --> "冲突检测器"
    "冲突检测器" -->|"检测到冲突"| "异步冲突队列"
    "冲突检测器" -->|"无冲突"| "联动执行器"
    "异步冲突队列" --> "仲裁规则引擎"
    "仲裁规则引擎" --> "解析策略匹配"
    "解析策略匹配" --> "联动执行器"
    "联动执行器" --> "闭环追踪器"
    "闭环追踪器" --> "IndexedDB 快照存储"
    "IndexedDB 快照存储" --> "状态时间线"
    "闭环追踪器" --> "监控仪表盘"
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深蓝黑底色 (#0a0e1a) + 青蓝科技光 (#00e5ff) + 琥珀告警色 (#ff9100) + 翠绿正常色 (#00e676)
- **按钮风格**：微圆角（4px），玻璃拟态效果，hover 时光晕扩散
- **字体**：数据展示用 JetBrains Mono，UI 文本用 Noto Sans SC，标题用 Orbitron
- **布局风格**：左侧固定导航栏 + 顶部状态栏 + 主内容区网格布局
- **图标风格**：线性科技风，2px 描边，发光效果

### 4.3 响应式设计

- 桌面优先设计，最小支持 1280px 宽度
- 平板端：侧边栏折叠为抽屉，仪表盘网格降为双列
- 移动端：单列布局，底部导航切换页面

### 4.4 3D 场景指引

- 户型图热力图可考虑 3D 俯视渲染，但优先使用 2D Canvas 保证性能
- 设备状态矩阵使用 CSS 3D 变换实现微透视效果
